This is a blog built with Bootstrap.

Me:

Hi! I'm Iris. I built this site. In case you cannot tell, it uses Bootstrap. This was all part of an effort to understand how Bootstrap works and I think I've accomplished that. My project approach consisted of the following:

Reading the documentation, re-reading the documentation, and reading the documentation again. The goal of these first reads were to really just understand how a CSS framework functioned at a high level. It also helped me learn what exactly I needed and didn't need to download. I also read outside articles to determine which version to download and specificities to know about using the service.

Setting up files and Github. Took longer than I expected.

Creating the main components of my site (navigation and hero banners). This required more reading of the Bootstrap documentation.

Creating the rest of my site. Had to shuffle some things around.

Hope you guys enjoy!
